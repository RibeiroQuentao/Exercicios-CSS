Demonstração de CSS Externo

Note que nesse caso não existe nenhum codigo CSS na página em HTML, pois o arquivo que contém as formatações é externo. Para isso vo^Ce deve criar um arquivo .CSS em uma pasta "Style" e indicá-lo através de um link dentro de HEAD.

Nesse caso todos os parágrafos irão herdar as características definidas pelo o arquivo .CSS, note que qualquer arquivo HTML que você criar e indicar o arquivo CSS externo será formado pelo padrão definido, isso é muito útil para projetos com mais de um arquivo em HTML.
