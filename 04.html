Seletores

Seletor de TAG

Ao usar a seletor tipo tag em H1 e H2 todos os elementos deste tipo vão herdar os estilos definidos pelo código CSS

Seletor tipo .id

O seletor tipo #ID deve ser usado uma única vez no arquivo! Cada elemento pode ter apenas um ID. Cada página pode ter apenas um elemnto com aquele ID.

Seletor de Classe

Qualquer informação do estilo que precise ser aplicada a múltiplos elementos em uma página deve ser feita com uma classe. Você pode usar a mesma classe para vários elementos. Você pode usar a mesma classe para vários elementos. Você pode usar várias classes para um mesmo elemento.
